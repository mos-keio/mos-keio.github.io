パスワードの変更方法について

NIS+では、ログインパスワード以外に、SecureRPCパスワードが使用されます。
このSecureRPCパスワードは、ログインパスワードの変更やログインシェルの変更を 行う際に必要になります。
ログインパスワードを変更するには，1の手順でログインパスワードと SecureRPCパスワードを一致させてから2の手順でログインパスワードを変更して ください．

ここに書かれている作業は全て，NIS+サーバ(= tokyo)にログインして行なってください．

1. ログインパスワードとSecureRPCパスワードを同一にする手順

この手順を実行することで、SecureRPCパスワードをログインパスワードと一致させることが 出来ます。
一度、実行しておけば、ログインパスワードの変更と同時に、自動的にSecureRPCパスワード も同じものに変更されます。

手順

  1. chkey -pコマンドを実行
  2. デフォルトSecureRPCパスワード(nisplus)を入力
  3. ログインパスワードを入力

実行例

$ chkey -p[ENTER]
Updating nisplus publickey database.
Reencrypting key for 'unix.ユーザID@mos1.mos.ics.keio.ac.jp'.
Please enter the Secure-RPC password for ユーザ名: nisplus[ENTER]
Please enter the login password for ユーザ名: ログインパスワード[ENTER]

ユーザIDは"id ユーザ名"を実行すると"uid="の後に書かれている数字です.

2. パスワードの変更

ログインパスワードを変更する場合は、1の手順を実施した後に、実行してください。
その際に、ログインパスワードとSecureRPCパスワードは連動して変更されます。

手順

  1. keyloginコマンドを実行
  2. SecureRPCパスワード（＝ログインパスワード）を入力
  3. passwdコマンドまたはnispasswdコマンドを実行
  4. 対話環境でパスワードを変更

実行例

$ keylogin[ENTER]
Password:SecureRPCパスワード[ENTER]
$ nispasswd[ENTER]
Changing password for ユーザ名 on NIS+ server
Old NIS+ password:ログインパスワード[ENTER]
Last successful password change for ユーザ名: Thu Aug 9 12:50:49 2002
Last unsuccessful password change for ユーザ名: Thu Aug 9 12:53:22 2002

3. パスワードに関する注意事項(重要)

パスワードは以下の条件を全て満たすものを使うようにしてください．

  • 6文字以上
  • 大文字と小文字を織りまぜる
  • 記号を含める
  • 辞書に載っている単語(特に英単語)は使わない
  • ユーザ名の部分文字列を使わない
